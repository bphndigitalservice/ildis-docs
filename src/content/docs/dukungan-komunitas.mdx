Untuk mendapatkan bantuan, berbagi pengalaman, atau berdiskusi dengan pengguna / pengembang ILDIS dalam naungan Jejaring JDIHN, Anda dapat bergabung ke grup komunitas ILDIS.

Anda dapat berdiskusi terkait fitur, permasalahan, bug dan lainnya melalui :

-  <a href="https://github.com/bphndigitalservice/ildis/discussions">Github Discussion</a>

- **WhatsApp**: <a href="https://chat.whatsapp.com/KvUHasGEH4I4xwWHPjWzu8" target="_blank">Join</a>

Melalui grup ini, Anda bisa:

- Mendapatkan bantuan teknis dari sesama pengguna dan pengembang
- Berbagi ide pengembangan fitur baru
- Mendapatkan informasi pembaruan versi ILDIS
- Berdiskusi mengenai praktik terbaik pengelolaan dokumentasi hukum

Kami sangat mendorong partisipasi aktif dari seluruh anggota JDIHN untuk memperkuat kolaborasi dan mendukung pengembangan ILDIS secara berkelanjutan.

> ⚠️ **Catatan**:
> Untuk menjaga kualitas diskusi dan efektivitas dukungan, kami sangat menyarankan agar anggota yang bergabung paling tidak memiliki pengalaman atau kompetensi dasar dalam pengelolaan server dan/atau pengembangan web. ILDIS ditujukan untuk digunakan dan dikembangkan secara mandiri oleh anggota JDIHN yang memiliki infrastruktur sendiri.
