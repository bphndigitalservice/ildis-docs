Pilih metode instalasi ILDIS yang paling sesuai dengan kebutuhan Anda

1

Pasang Cepat (One-Click)

Instalasi tercepat dengan script otomatis. Memerlukan Docker/Podman.

Direkomendasikan
2

Docker

Container dengan image GHCR atau build dari source. Baik untuk production dan development.

3

Vagrant

Virtual machine untuk development. Termasuk semua dependensi terkonfigurasi.

Hanya Development
4

Manual

Konfigurasi lengkap pada server Anda sendiri. Linux/macOS (LAMP/LEMP) dan Windows (XAMPP).

Linux/macOS/Windows

Setelah Instalasi (After Installation Steps)

Setiap metode instalasi akan diikuti langkah-langkah post-installation:

STEP 1

Konfigurasi Environment (.env)

Salin dan sesuaikan file .env dengan database dan kredensial Anda.

STEP 2

Import Database

Import skema database awal untuk menjalankan aplikasi.

STEP 3

Konfigurasi Security

Generate Cookie Validation Key untuk keamanan session.

STEP 4

Buat User Admin

Buat user awal untuk akses backend menggunakan CLI.

STEP 5

reCAPTCHA (Opsional)

Konfigurasi Google reCAPTCHA v3 untuk halaman login admin.
